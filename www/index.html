ROS WEB Controller

TAMIYA Cam-Program Robot Manager

Default Teleoperator 

 roslaunch tamiya_cam_robot teleop.launch
This web server sends motor efforts std_msgs::String topics to a motor control node.

Teleoperator w/ Twist 

 roslaunch tamiya_cam_robot teleop_twist.launch
This web server sends moving efforts (linear and angular) as std_geometry::Twist topics to a motor control node.

Music Player 

 roslaunch tamiya_cam_robot music_player.launch
This web server sends a full path of a music file as std_msgs::String topics to a music player node.

LED Tuner 

 roslaunch tamiya_cam_robot led_tuner.launch
This web server sends hex-color codes as std_msgs::UInt32MultiArray topics to a color listener node.

LED Tuner 

 roslaunch tamiya_cam_robot curiosity.launch
This robot has 3 servo motors for head and arms.
